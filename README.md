Next Js Boilerplate

Project ini untuk setup awal Nextjs yang memiliki halaman frontend dan halaman backend (Admin)

Hal yang perlu diperhatikan adalah :
1. Setup .env.local
2. Setup logo logonya

## Cara Install Aplikasi
Saat pertama kali clone, jangan lupa untuk `npm install`

## Setting Logo Aplikasi
Untuk logo Aplikasi, harus diletakkan di folder 'public/images/logo'.
Logo yang harus ada adalah :
    - auth-logo.svg -> Logo untuk halaman autentikasi
    - logo-dark.svg -> Logo untuk tema dark
    - logo-icon.svg -> Logo yang berupa icon
    - logo.svg -> Logo utama
